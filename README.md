## What It Does

- Maintains a structured, versioned project repository  
- Tracks schedule dependencies and critical path changes  
- Organizes inspections, emails, and project documentation  
- Surfaces coordination gaps and emerging risks  

GC Companion is designed to reduce cognitive load on complex builds without replacing builder judgment.
